A New Start

2017-08-30 21:42:04 | 琐事高中

感觉自己终于快要从精神分裂症恢复过来了。

多巴胺是个神奇的神经递质，在睡眠不足时疯狂的分泌使得人极度的暴躁，在看动漫以至于听歌时可以进行神经和体液调节弄得头皮发麻浑身起鸡皮疙瘩。它能控制人心情变好与变坏，还能使人对某种他喜欢的东西上瘾。它分泌过多的时候会让人极度依赖某些他喜欢的东西，在它分泌不足是还能让人注意力不集中，胡思乱想什么的。

原来好多看似不顺心的事其实根本原因还是出在自己身上。
明知道自己做不完却还要给自己找那么多事情去做，干嘛非把自己的精力耗费在乱七八糟的小事上。

所以呢？还是要多睡觉，还是要接受这些事情，荨麻疹还是特别痒…

《你的名字》在B站上映了。
毫不犹豫的去花钱下载了正版。找个时间猫在屋里认认真真的再看一遍。
自己看电影比在班里集体看效果好多了，起码没人会在观看时毁这部电影。看了3遍了还是没能把剧情彻底看懂，时间线还是摸不清楚，整个电影里藏了好多细节我之前完全没注意到，看来还是自己太幼稚了看不懂这剧情。

但是不知怎么的就是越来越喜欢这个电影，对日漫有了完全不一样的看法。

对自己好一点，最起码还是要活着的，总不能昏昏沉沉的吧。
